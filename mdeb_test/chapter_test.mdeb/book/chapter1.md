## Chapter 1: Markdown Compliance

The headings below range from one to six hashtags

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6

These two setext headings use equals or hyphens underneath them. 

Heading 1
====

Heading 2
----

This is an underline

---

These are text alterations. 

This is *italics* with single asterisks, and _italics_ with single underscores

This is **bold** with double asterisks, and __bold__ with double underscores

This in_tra_word should not render as an itali, but rather plain text. The first underscore should have a space before it, and the last should have a space after for it to be valid italics. 

This is ***both*** with triple asterisks, and ___both___ with triple underscores. 

These are lists in Asterisks

* Item 1
* Item 2
* Item 3

These are lists in Pluses

+ Item 1
+ Item 2
+ Item 3

These are lists in Hyphens

- Item 1
- Item 2
- Item 3

These are ordered lists with periods

1. Item 1
2. Item 2
3. Item 3

These are links. 

This is a [link to GitHub](https://github.com).
